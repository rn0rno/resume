## 意欲・興味
### 得意なこと
- あるミッションを達成するためにできることを考え、実行することが得意です
  - 情報を集めたり、個人的な見解を沿えて意思決定することが得意です
- 業務フローやプロジェクト進行の効率化や見える化、言語化、問題の切り分けが得意です
- サーバーサイドの開発がメイン領域ですが、インフラ (クラウド）にも比較的明るいです
- チームの立ち上げや人の育成は得意です

### 苦手なこと
- ステークホルダーがたくさんいて、なかなか意思決定ができない・意思決定する軸や人が明確ではない状況ではうまくワークしません
- 意思決定に対しての納得度が自分の中で作れないものを実行することは苦手です
